kill В· killall В· pkill

$ kill 1234 # send SIGTERM (15) вЂ” graceful stop
$ kill -9 1234 # send SIGKILL вЂ” force kill, no cleanup
$ kill -SIGTERM 1234 # same as kill 1234 (by name)
$ killall firefox # kill ALL processes named firefox
$ killall -9 firefox # force kill all firefox processes
$ pkill httpd # kill by name pattern
$ pkill -u ali # kill ALL processes owned by ali
Rule: Always try SIGTERM (15) first вЂ” gives process time to clean up. Use SIGKILL (9) only if it doesn't respond.

Jobs вЂ” Foreground & Background

$ sleep 300 & # run in BACKGROUND from start [1] 4321 в†ђ job# and PID
$ jobs # list all background jobs [1]+ Running sleep 300 & [2]- Stopped vim file.txt
$ fg # bring latest job to FOREGROUND
$ fg %1 # bring job #1 to foreground
# While a process is running: Ctrl+Z в†’ suspend (stop) it Ctrl+C в†’ terminate it (SIGINT)
$ bg # resume suspended job in BACKGROUND
$ bg %2 # resume job #2 in background
$ nohup script.sh & # run immune to hangup (survives logout) Output: nohup.out
Flow: run command в†’ Ctrl+Z to suspend в†’ bg to push background в†’ fg to bring back

$ nice -n 10 ./backup.sh # start with low priority (nice 10)

Note: Regular users can only increase nice (lower priority). Only root can decrease nice (raise priority).
