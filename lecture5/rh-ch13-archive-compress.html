Memory trick: create · extract · list · filename — always use f last before the filename!
Order matters: flags come before the archive name. tar -czf backup.tar.gz files/ — the f flag must be immediately before the filename.
📦

tar — Creating Archives

No Compression (.tar)
$ tar -cf archive.tar file1 file2 # archive files only — no compression
$ tar -cf backup.tar /etc/ # archive entire /etc directory
gzip (.tar.gz)
$ tar -czf backup.tar.gz /etc/ # create + gzip compress
$ tar -czvf backup.tar.gz /etc/ # same + verbose (show files)
bzip2 (.tar.bz2)
$ tar -cjf backup.tar.bz2 /etc/ # create + bzip2 compress
xz (.tar.xz) — best compression
$ tar -cJf backup.tar.xz /etc/ # create + xz compress (best ratio)
Multiple sources
$ tar -czf backup.tar.gz /etc/ /home/ /var/log/ # archive multiple dirs at once
📂

tar — Extracting Archives

$ tar -xf archive.tar # extract to CURRENT directory
$ tar -xzf backup.tar.gz # extract gzip archive
$ tar -xjf backup.tar.bz2 # extract bzip2 archive
$ tar -xJf backup.tar.xz # extract xz archive
$ tar -xzf backup.tar.gz -C /tmp/restore/ # extract to SPECIFIC directory
$ tar -xzf backup.tar.gz etc/hosts # extract ONE specific file only
-C flag = Change to directory before extracting. Always use it to avoid cluttering your current directory!
🔍

tar — Listing Contents

$ tar -tf archive.tar # list contents WITHOUT extracting etc/ etc/hosts etc/passwd etc/ssh/sshd_config
$ tar -tzf backup.tar.gz # list gzip archive contents
$ tar -tvf archive.tar # list with details (permissions, size, date) -rw-r--r-- root/root 158 2026-03-07 etc/hosts -rw-r--r-- root/root 1234 2026-03-07 etc/passwd
$ tar -tzvf backup.tar.gz | grep passwd # search for specific file in archive

gzip · bzip2 · xz — Standalone Commands

gzip
$ gzip file.txt # → creates file.txt.gz # original file is REMOVED
$ gzip -k file.txt # keep original file
$ gunzip file.txt.gz # decompress
$ gzip -d file.txt.gz # same as gunzip
$ zcat file.txt.gz # read gz file without extracting
bzip2
$ bzip2 file.txt # → creates file.txt.bz2 # original file is REMOVED
$ bzip2 -k file.txt # keep original file
$ bunzip2 file.txt.bz2 # decompress
$ bzip2 -d file.txt.bz2 # same as bunzip2
$ bzcat file.txt.bz2 # read bz2 file without extracting
xz
$ xz file.txt # → creates file.txt.xz # original file is REMOVED
$ xz -k file.txt # keep original file
$ unxz file.txt.xz # decompress
$ xz -d file.txt.xz # same as unxz
$ xzcat file.txt.xz # read xz file without extracting
Warning: gzip, bzip2, and xz delete the original file by default! Use -k flag to keep the original.
🤐

zip & unzip

$ zip archive.zip file1 file2 # zip files together
$ zip -r archive.zip mydir/ # zip entire directory (-r recursive)
$ unzip archive.zip # extract zip archive
$ unzip archive.zip -d /tmp/restore/ # extract to specific directory
$ unzip -l archive.zip # list contents without extracting
zip keeps the original files — no need for -k. Best when sharing with Windows users.
🚀

Archive + Transfer Over SSH

Create then copy
# step 1 — create archive $ tar -czf /tmp/backup.tar.gz /etc/
# step 2 — copy to remote server $ scp /tmp/backup.tar.gz ali@server:/tmp/
# step 3 — extract on remote $ ssh ali@server 'tar -xzf /tmp/backup.tar.gz -C /restore/'
Stream directly over SSH (no temp file)
# compress + send in ONE command $ tar -czf - /etc/ | ssh ali@server 'tar -xzf - -C /restore/' # - means stdin/stdout (pipe it directly)
# receive archive FROM remote $ ssh ali@server 'tar -czf - /etc/' | tar -xzf - -C /local/restore/

Exam Tips

⚠ DELETEgzip, bzip2, xz delete the original file after compressing! Use -k to keep it. zip does NOT delete originals.
🔑 FLAGS-z=gzip · -j=bzip2 · -J=xz. Note capital J for xz — easy to confuse with lowercase j for bzip2!
✔ LIST FIRSTAlways run tar -tf archive.tar.gz before extracting — check what's inside and where it will land.
📁 -C FLAGUse -C /path/ to extract into a specific directory instead of your current location — avoids messy extractions.
⚠ f FLAGThe f flag must come immediately before the archive filename. Wrong: tar -fczv → Right: tar -czvf archive.tar.gz
💡 SPEEDIn a hurry? Use gzip (-z). Need smallest file? Use xz (-J). Need Windows compatibility? Use zip.
✔ PRESERVEUse tar -p when archiving system files — preserves original permissions and ownership on extraction.
🔍 IDENTIFYDon't know archive type? Use file archive.tar.gz — it reads the magic bytes and tells you the real format.
